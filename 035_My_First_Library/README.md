# 📦 My First C++ Library Project

## 📌 Overview

This project is a simple implementation of a custom C++ utility library created for learning purposes.
It demonstrates how to organize reusable code using **namespaces**, **header files**, and **modular design**.

The main idea is to build small libraries that can be reused in different projects instead of rewriting the same code every time.

---

## 🎯 Project Goal

* Learn how to structure C++ code into reusable modules
* Practice separating logic into different header files
* Understand the concept of utility libraries
* Apply basic software design principles in C++

---

## 🧠 Libraries Included

### 🔹 InputLib

Handles user input operations:

* Read numbers
* Read characters
* Read strings
* Read positive numbers

---

### 🔹 OutputLib

Handles all output operations:

* Print names
* Print integers
* Print floats
* Print characters

---

### 🔹 ValidationLib

Handles input validation:

* Validate integer input
* Validate character input

---

### 🔹 MyMathLib

Basic mathematical operations:

* Half number calculation
* Grade checking
* Swapping two numbers
* Range validation

---

### 🔹 CheckNumLib

Number checking utilities:

* Check even or odd numbers
* Check perfect numbers

---

## 🧪 Example Usage

The `main.cpp` file demonstrates how to use all libraries together:

* Reading user input
* Printing output
* Performing mathematical operations
* Checking number properties
* Swapping values
* Validating ranges
